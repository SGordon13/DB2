To install and run:
Using XAMPP and phpMyAdmin:
1. start MySQL and Apache from XAMPP.
2. Create a new database named db2.
3. run the sql file named DB2.sql on your db2 database.
	-This creates all the tables and runs the insert statements.
4. Open 'http://localhost/dashboard/DB2/SignIn.php' in your browser.
